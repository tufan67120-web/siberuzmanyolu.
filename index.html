Siber Güvenlik & Etik Hacker Akademisine Hoş Geldiniz

Bu site tamamen eğitim amaçlıdır. Yasadışı hack öğretilmez; yalnızca savunma, anonimlik ve etik hacking bilgileri sunulur.

1. Anonim Nasıl Kalınır? (YASAL)

  • VPN kullanımı
  • TOR Browser
  • Güvenli tarayıcı ayarları
  • Çerez/iz temizleme
  • IP gizleme teknikleri
  • Telefon ve bilgisayarda gizlilik ayarları

2. Etik Hack Öğrenme Aşamaları

  1. Temel ağ bilgisi (IP, DNS, port, paketler)
  2. Linux öğrenme (Kali Linux, Parrot OS)
  3. Ağ tarama araçları (Nmap, Netdiscover)
  4. Zafiyet analiz araçları (Nikto, OpenVAS)
  5. Web güvenliği (SQLi, XSS mantığını öğrenme)
  6. Sızma testi raporlaması

3. Beyaz Şapkalı Hacker Nasıl Olunur?

  • Resmi eğitimler almak
  • Sertifika: CEH, OSCP
  • Sızma testi çalışmaları
  • Bug Bounty programlarına katılma

4. Gri Şapkalı Hacker Nedir?

Ne iyi ne kötü. Açıkları bulur ama bazen izinsiz test yapabilir. YASAL DEĞİLDİR, sadece tanım eklenmiştir.

5. Siyah Şapkalı Hacker Nedir?

Sistemlere izinsiz giren kişidir. Sitede anlatılmaz, sadece tanım olarak belirtilir.

6. Yasal Uyarı

Bu sitedeki tüm bilgiler savunma ve eğitim amaçlıdır. İzinsiz işlem yapmak suçtur.

7. Web Sitesi Nasıl Kurulur?

  1. Domain (alan adı) seçilir.
  2. Hosting (sunucu) alınır veya ücretsiz GitHub Pages kullanılır.
  3. index.html dosyası oluşturulur.
  4. HTML, CSS, JS ile sayfa tasarlanır.
  5. Dosyalar sunucuya yüklenir. (FTP veya GitHub)

8. Hacker Saldırıları Nasıl Çalışır? (YASAL BİLGİ)

  • Phishing: Sahte mail/sayfa ile şifre çalma (nasıl yapıldığı öğretilmez).
  • Brute Force: Şifre tahmin etme mantığı.
  • SQL Injection: Veri tabanı sorgularının yanlış kullanılması.
  • DDoS: Aşırı trafik gönderme.
  • MITM: Araya girerek veriyi dinleme.

9. Siber Güvenlik Uzmanı Ne Yapar?

  • Zafiyet taraması yapar.
  • Firewall ve ağ güvenliği ayarları yapar.
  • Saldırı tespiti için log inceler.
  • Penetrasyon testi gerçekleştirir.
  • Web güvenliği analizi yapar.

10. Hackerlara Karşı Kendini Nasıl Korursun?

  • Güçlü şifre + 2FA kullan.
  • VPN olmadan ortak Wi-Fi'ye bağlanma.
  • Şüpheli linklere tıklama.
  • Uygulamaları güvenilir yerden indir.
  • Cihazları güncel tut.
  • Gizlilik ayarlarını sıkılaştır.
  • Bilmediğin dosyaları açma.

11. Hacker Nasıl Olunur? (ETİK – YASAL)

  1. Linux öğren.
  2. Ağ temelleri: IP, port, paket mantığı.
  3. Nmap ile ağ taramayı öğren.
  4. Açık tarama araçlarını incele.
  5. Web güvenliği temelleri: XSS, SQLi mantığı.
  6. CTF’lere katıl.
  7. CEH veya OSCP sertifikası al.
